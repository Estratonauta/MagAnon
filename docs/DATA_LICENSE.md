# Public data pool license

MagAnon code is MIT licensed.

The **public data pool** (crowd-sourced magnetic survey samples and derived aggregates) is licensed under the **Open Database License (ODbL) v1.0**.

## Why ODbL
- Designed for shared datasets / databases.
- Allows commercial and non-commercial use.
- Requires **attribution**.
- Requires **share-alike** for publicly shared derivative databases, so improvements to the database can remain open.

## Notes
- Individual contributors can export their own session data for personal use.
- Uploading to the public pool is opt-in; uploaded data is contributed under ODbL.
